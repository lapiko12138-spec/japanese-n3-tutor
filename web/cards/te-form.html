🔄 て形变化图卡

理解发音逻辑 + 锚点动词,别靠死记 · て形是 ています/てください/た形 的共同基础

① 先挑出简单的两类

Ⅱ类(一段)
去る + て
食べる→食べて · 見る→見て · 寝る→寝て
Ⅲ类(不规则 · 背)
背下来
する→して · 来る→来て(きて)

② Ⅰ类:看词尾分 5 组(颜色分组)

词尾 う・つ・る
→ って
買う→買って · 待つ→待って · 帰る→帰って
⚓ 锚点:買う(买)→ 買って
词尾 む・ぶ・ぬ
→ んで
読む→読んで · 遊ぶ→遊んで · 死ぬ→死んで
⚓ 锚点:飲む(喝)→ 飲んで
词尾 く
→ いて
書く→書いて · 聞く→聞いて
⚓ 锚点:書く(写)→ 書いて
词尾 ぐ
→ いで
泳ぐ→泳いで · 急ぐ→急いで
⚓ 锚点:泳ぐ(游)→ 泳いで
词尾 す
→ して
話す→話して · 貸す→貸して
⚓ 锚点:話す(说)→ 話して
⚠️ 唯一例外
行く → 行って
行く 是Ⅰ类,本该→行いて,但它特殊,记 行って(不是行いて)

③ 为什么这么分?(理解就不用背)

  • う・つ・る → って:念念看,嘴都是"啪"地干脆收尾 → 顿一下的小促音 っ。
  • む・ぶ・ぬ → んで:念念看,都要闭嘴/鼻子哼一下(m·b·n 鼻音浊音)→ ん,并把て浊化成 で。
  • く → いて / ぐ → いで:k 软化成 い;ぐ是浊音所以 て→で。
  • す → して:す→し(像ます形那样滑到い段)+ て。
🎵 て形之歌(带节奏念几遍)
う・つ・る → って!　む・ぶ・ぬ → んで!
く → いて、ぐ → いで、す → して!
行く は 行って(例外!)

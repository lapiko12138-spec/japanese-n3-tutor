## 步骤4+5 · 逐块授课(讲一块→课中测试一块)

### 知识块 1:什么是辞书形
- **概念**:动词的「原形 / 基本形」,字典里收录的形态。最朴素、最短。
- **结构特征**:所有辞书形都以 **う段假名** 结尾:う・く・ぐ・す・つ・ぬ・ぶ・む・る。
- **场景**:它是「素颜」状态,礼貌的「ます形」是化了妆。和朋友、自言自语、字典、语法书里用辞书形。
- **例句**:毎日ご飯を**食べる**(每天吃饭) / 本を**読む**(读书)。
- **常见错误**:把「食べます」当原形。ます形是变出来的,不是原形。
- **对比**:食べる(辞书形/简体)↔ 食べます(ます形/礼貌)。

**课中测试 1**(判断,2 题,即时反馈):
- a) 下列哪个是辞书形:① 書きます ② 書く → 答:②(書く)
- b) 「見る・見ます」中辞书形是哪个? → 答:見る

### 知识块 2:为什么必须区分动词类别
- **核心理由**:日语动词变形(ます/て/た/ない/可能/被动/使役…)的规则**因类别而不同**。不先分类,后面每一种变形都会错。
- **类比**:像英语动词分规则/不规则;不知道类别就背不对变化。
- **三类命名**(多种叫法,统一即可):
  - Ⅰ类 = 五段动词 = u-verb(godan)
  - Ⅱ类 = 一段动词 = ru-verb(ichidan)
  - Ⅲ类 = 不规则 = する・来る
- **常见错误**:以为「以る结尾就是Ⅱ类」——错,很多る结尾是Ⅰ类(帰る)。

**课中测试 2**(理解,1 题):
- "为什么学变形前必须先会分类?用你自己的话说一句。" → 期望答到「不同类变形规则不同」。

### 知识块 3:Ⅰ类(五段)动词
- **判别**:
  1. **不以る结尾** → 必为Ⅰ类:書**く**・読**む**・話**す**・行**く**・買**う**。
  2. 以る结尾,但 **る前是 a/u/o 段音** → Ⅰ类:分か**る**(ka)・作**る**(ku)・乗**る**(no)。
- **名称由来**:词尾可在五个段(あ/い/う/え/お)上变化,故「五段」。
- **例**:書く=Ⅰ类;読む=Ⅰ类;話す=Ⅰ类。
- **常见错误**:看到「帰る」就判Ⅱ(下一块专门讲例外)。

**课中测试 3**(分类,3 题):书く / 読む / 話す 各是几类? → 全Ⅰ类。追问:为什么?(不以る结尾)

### 知识块 4:Ⅱ类(一段)动词
- **判别**:以 **る** 结尾,且 **る前是 i 段或 e 段音** → 多为Ⅱ类:
  - 食べ**る**(be=え段)・見**る**(mi=い段)・起き**る**(ki=い段)・寝**る**(ne=え段)。
- **名称由来**:词尾只在一个段上变化,故「一段」。
- **例**:食べる・見る・起きる 都是Ⅱ类。
- **对比**:同是る结尾,作る(ku→う段)是Ⅰ,食べる(be→え段)是Ⅱ。**看る前那个音在哪一段**是关键。

**课中测试 4**(分类+理由,3 题):食べる / 見る / 起きる 各几类?为什么? → 全Ⅱ类,る前是 い/え 段。

### 知识块 5:Ⅲ类(不规则)+ 常见例外
- **Ⅲ类只有两个**:**する**(做)、**来る(くる)**(来)。变形不规则,直接背。
  - 「名词+する」也算:勉強する・運動する。
- **★ 常见例外(看似Ⅱ类,实为Ⅰ类)**——必须背:
  - **帰る(かえる)**・入る(はいる)・走る(はしる)・要る(いる)・知る(しる)・切る(きる)・限る(かぎる)。
  - 它们る前是 い/え 段(像Ⅱ类),但实际按Ⅰ类变形。
- **常见错误**:把 帰る 当Ⅱ类 → 后面ます形会错成「帰ます」(误)。正确:帰**り**ます。

**课中测试 5**(2 题):
- a) する 是几类? → Ⅲ类。
- b) 帰る 是几类?它的特殊在哪? → Ⅰ类(例外),る前是え段却按Ⅰ类变。

### 知识块 6:辞书形 → ます形(初步)
- **规则**:
  - **Ⅰ类**:词尾 **う段 → 同行い段 + ます**。
    - 書く→書**き**ます / 読む→読**み**ます / 話す→話**し**ます / 帰る→帰**り**ます。
  - **Ⅱ类**:**去る + ます**。
    - 食べる→食べます / 見る→見ます / 起きる→起きます。
  - **Ⅲ类**:する→**します** / 来る→**来(き)ます**。
- **场景**:对外、礼貌、正式场合用ます形。
- **常见错误**:Ⅰ类直接去る(误「書ます」);忘了变段(误「書くます」)。

**课中测试 6**(变形,4 题,即时反馈):
- 読む→? (読みます) / 食べる→? (食べます) / 帰る→? (帰ります) / する→? (します)

## 步骤8 · 课后测试题库(14 题,答案仅老师可见)

**第二层 · 规则应用(6 题)**

7. 辞书形→ます形:見る → (見ます)
8. 辞书形→ます形:話す → (話します)

10. 改错:帰る的ます形写成「帰ます」对吗?改正。→ (错;帰る是Ⅰ类例外,应为「帰ります」)

**第三层 · 主动输出(4 题)**
